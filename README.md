# Avanti Buche – AI Engineer Portfolio

A beautiful, dark-themed Streamlit portfolio built from resume data.

## 🚀 Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

Then open http://localhost:8501 in your browser.

## 🌐 Deploy for Free on Streamlit Community Cloud

1. Push this folder to a **GitHub repository** (can be public or private).
2. Go to https://share.streamlit.io and sign in with GitHub.
3. Click **"New app"** → select your repo → set `app.py` as the main file.
4. Click **Deploy** – your public URL will be ready in ~1 minute!

Your portfolio will be live at:
`https://<your-github-username>-portfolio-<random>.streamlit.app`
